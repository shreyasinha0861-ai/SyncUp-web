SyncUp is a fully functional social media website designed specifically for students to find project partners and collaborate on tech projects. This project was built for the Techphilia 9.0 technical festival.

Frontend: HTML5, CSS3 (Custom Variables), Google Fonts (Inter & Pacifico)

Styling Framework: Responsive Design for mobile & desktop

Backend & Database: Firebase (Firestore Cloud Database)

Authentication: Firebase Auth (Email/Password)

Deployment: [Mention if you used Firebase Hosting/GitHub Pages

Key Features
User Authentication: Secure Register and Login system using Firebase Auth.

Dynamic Profile Management: Users can update their name, expertise (e.g., Java, HTML), and upload a professional profile picture.

Global Project Feed: A real-time feed where students can post their project ideas with images.

Interactive Engagement: * Real-time Likes: Instant updates using Firebase Transactions.

Comment System: Students can discuss projects directly on posts.

Collaborator Community: A dedicated group chat space for real-time networking.

Responsive UI: Fully optimized for all screen sizes, featuring a modern sidebar and glassmorphism-inspired modals.# SyncUp-web
